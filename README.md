# Employee-Management-System

A **web application** to manage railway employee records in real-time, built with **React + Vite**, **Tailwind CSS**, and **MongoDB**.

## Features

- Add, view, update, and delete employee records.
- Search and filter employees by name, role, or department.
- Responsive design using Tailwind CSS.
- Data stored securely in MongoDB.
- Fast and modern frontend with Vite + React.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Version Control:** Git & GitHub
